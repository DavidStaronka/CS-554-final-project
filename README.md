# CS-554-final-project

To run the server:
    cd into ./backend
    npm i
    npm run seed (to seed the example user)
    npm start

To run the socket.io server for DM interaction:
    cd into ./socket_server
    npm i
    npm start

To run the client:
    cd into ./client
    npm i
    npm start

Navigate to http://localhost:3000 to open the website
